# Xenon Client

A Minecraft Fabric client mod featuring a circular wheel GUI for organizing sections — inspired by Meteor Client but with a unique radial interface.

## Features

- **Radial Wheel GUI** — Press `Right Shift` to open a sleek circular menu
- **Section Categories** — Combat, Movement, Render, Player, World, Misc
- **Section Panels** — Click a section to open its detail panel
- **Animated UI** — Smooth open/close animations with easing
- **Fancy Styling** — Glowing center, color-coded sections, hover effects

## Requirements

- Minecraft 26.1.1
- Fabric Loader 0.18.4+
- Java 25+

## Building

```bash
./gradlew build
```

The built jar will be in `build/libs/`.

## Usage

1. Install the mod jar into your `.minecraft/mods` folder
2. Launch Minecraft with Fabric Loader
3. Press **Right Shift** to open the Xenon wheel GUI
4. Hover over sections and click to open their panels
